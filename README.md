In this lab I will be securing my home network, creating a secure network to showcase personal development and skills, and finally to explore and enjoy the beautiful world of tech. 
# Home Network Infrasructure Project

## Overview
Designed and upgraded my home netowrk using a phased implementation strategy with the TP-Link Archer c54 & Tp-Link ER605. 

## Phase I : Implementation & Stability 
- Installed and configured TP-Link Archer router
- Established reliable WAN/LAN connectivity
- Configured SSIDs, WPA security, and firmware updates.
- Tested baseline speeds and device coverage

## Phase II: Security & Management 
- Added TP-Link ER605 gateway
- Configured DHCP reservations and NAT
- Created firewall/access rules
- Improved traffic control and device management
- Began document topology and settings

## Phase III Expansion & Eneterprise Features 
-Planned VLAN segmentation for trusted / guest / IoT devices 
-Planned VPN remote access for secure administration 
-Planned monitoring/logging for troubleshooting 
-Designed scalable layout for future switches/access points 

## Phase IV Services, Storage, & Automation (Planned) 
###Service Deployment
- Deploy home server for internal services (e.g., web hosting, monitoring dashboard)
- Implement network monitoring tools (uptime, bandwidth usage)
- Explore containerization (Docker) for service management
  
###Storage & Infrastructure 
- Set up network-attached storage (NAS) for centralized file management 
- Configure backups and redundancy
- Plan secure file access across devices

### AUtomation & coding 
-Develop Python scripts for network monitoring and alerts 
-Automate device status checks and logging 
-Explore API intergration with netowrok hardware 

##Long Term Goal
Transform home network into a small-scale enterprise lab environment simulating real-world IT infrastructure.
